# Quiz_de_Perguntas 🎮

Projeto acadêmico desenvolvido com **React Native** e **Expo**, com o objetivo de auxiliar estudantes iniciantes no aprendizado de programação por meio de um sistema de quiz gamificado.

## 🚀 Tecnologias
- React Native
- Expo
- JavaScript

## 📦 Como rodar localmente
git clone https://github.com/FelipeFaria01/Quiz_de_Perguntas.git
cd Quiz_de_Perguntas
npm install
npx expo start

## 📱 Download do App

Baixe agora o **Quiz_de_Perguntas 🎮** e comece a testar seus conhecimentos de programação de forma divertida e interativa!

👉 [Clique aqui para baixar o APK](https://expo.dev/artifacts/eas/qGjKqs7m39jQciti5QmKob.apk)

Instale no seu celular Android e aproveite.
